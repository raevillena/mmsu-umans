#1️ Frontend Framework & Libraries
    - React 19 – Latest core frontend library.
    - Vite – Modern, fast build tool and dev server (instead of CRA).
    - Material UI (v6.4.3) – For UI components and theming.
    - Emotion (v11.14) – CSS-in-JS styling used by MUI.
    - React Router v7 – Navigation and route handling.
    - Redux Toolkit v2.5 – Global state management.
    - React Redux v9.2 – Redux bindings for React.
    - Axios – For API requests.
    - React Hook Form – Lightweight form handling.
    - Yup & Zod – Schema validation libraries for forms and API responses.
    - Chart.js & react-chartjs-2 – For visualizing data via charts.
    - React Syntax Highlighter – Code display/highlighting.

#2️ Authentication & Security
    - Session-based Auth with Redis – Server maintains sessions instead of stateless JWT.
    - Google OAuth2 Support – via @react-oauth/google for Google sign-in integration.
    - Refresh Token Handling – Done in the frontend (renew on expiry).
    - RBAC – Role-Based Access Control (admin/user-level access).
    - CSRF Protection & Secure Cookies – Handled through backend with cookie settings.

#3️ API Communication
    - Custom Axios Wrapper – For consistent API handling.
    - Global Error Interceptors – API errors mapped to user-friendly messages.
    - Loading States & Toast Notifications – Enhanced UX on network activity and errors.

#4️ Form Handling & Validation
    - React Hook Form – Optimized performance form state management.
    - Yup – Declarative schema validation.
    - Zod – TypeScript-friendly schema validation (optional alternative or for API responses).

#5️ Project Structure Overview

/src
  /config       → Configuration files (e.g., axios instance, mqtt config) 
  /api          → Axios instance, authApi, userApi
  /components   → Reusable components (Navbar, Sidebar, etc.)
  /hooks        → Custom React hooks (e.g., useAuth, useFetch)
  /layouts      → Admin and user layouts
  /pages        → Auth, Dashboard, 404, etc.
  /store        → Redux Toolkit slices and store
  /utils        → Helpers like formatDate, validation logic
  /styles       → MUI Theme configuration
  App.jsx       → Main app component
  main.jsx      → Entry point
  routes.jsx    → Route definitions
  index.css     → Global styles

#6️ Tooling & Developer Experience
    - ESLint + Plugins – Code linting with React and hooks support.
    - TypeScript Support via @types/ – For typed development (optional if .ts not used yet).
    - vite-plugin-svgr – SVG imports as React components.
    - rollup-plugin-visualizer – Bundle analyzer for performance insights.
    - Environment Variables – .env for base API URLs, secrets, etc.
